Demonstration and Tutorial of WriteInTreeTutorialHello World - your first written-in-tree book

Create a folder for your future document.

Create a mindmap file in the document folder with the name "tree.mm", and with some arbitrary content, just a simple node like "Hello World" suffices.

Create the separate properties file in the document folder.

It is a configuration of your project.

This file has the name "properties.simco.text".

It uses the SimCo language.

The application can generate for you a default version of this file. I recommend you to use it.

Compile.

Call this compiler with the command line: "write-in-tree.exe --web -i <path-to-the-document-folder> -o <path-of-the-output-folder>"

Advice: put this command line into a script file for saving and possible later extensions. This script file in this tutorial will be referred to as the "compile script".
