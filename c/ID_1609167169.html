On Windows the installation of these 2 software is very easy. Most Linux distributions probably have packaged them.

Recommended Configuration

While they are promtly usable after installation, based on my own experience I suggest the following configuration right after installation.

In case of Freeplane:

In "Environment" -> "Save"

"Save folding" := "Never"

"Save used node-IDs only" := false

"Save modification times" := false

These changes make version control easier.

In case of Freemind there are similar settings.
