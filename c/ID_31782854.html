You Want to Use the Keyboard

While you can use mouse intuitively in these applications, you really want to use keyboard, because it is nicely supported and provides super-fast editing ability.

You can edit the text inside a node by F2.

You can fold, unfold nodes by the space key.

You can selected several nodes by holding down Shift.

You can move selected nodes by Ctrl + <arrow keys>.

You can insert new empty nodes by Enter, Shift + enter, Insert, Shift + Insert.

Rich formatting

As these applications are more than just outline editors, they allow rich formatting inside text nodes. But this compiler can not parse those correctly. So do not use this feature of your mindmap editor.

The compiler will error and report such situations.

In Freeplane you can easily convert a rich formatted node into a plain formatted one by pressing Alt+P.
